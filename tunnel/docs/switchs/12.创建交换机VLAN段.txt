消息：APICreateSwitchVlanMsg

权限：category = SwitchConstant.ACTION_CATEGORY, names = {"create"}, adminOnly = true

参数（默认必填）：
    逻辑交换机：要添加VLAN段的逻辑交换机
    起始VLAN：1~4094
    结束VLAN：1~4094

请求消息体示例：
{
    "com.syscxp.header.tunnel.switchs.APICreateSwitchVlanMsg":{
        "switchUuid":"860b3ee3dd1a4fb08ae2a48b30ecabaa",
        "startVlan":500,
        "endVlan":1000,
        "session":{
            "uuid":"b43c47795e84405ebfd3e455d8e6e158"
        }
    }
}


后台验证（validate）：
    1.起始VLAN要大于结束VLAN
    2.同一个物理交换机下的VLAN段不能有交叉
        如果该物理交换机是SDN交换机，则他上联交换机下的所有VLAN不能有交叉
        如果该物理交换机是MPLS交换机，则他下面所有SDN交换机的所有VLAN不能有交叉
