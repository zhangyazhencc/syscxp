任务需求：
1. 新购未付3天后自动删除
2. 过期3天后关闭（若能关闭）
3. 过期10天后删除
4. 过期后产品只能进行 删除、续费。其它操作不允许
5. 过期产品续费成功后产品变为正常，过期时间先按原来过期时间为起点延续。并按产品状态来判断产品是否需要下发开通
6. 接口有未删除专线使用不删除。
7. 专线新购支付成功，未开通前 过期时间为null


产品购买流程：
1. 先查询billing金额够不够。不够2，够3
2. 金额不够 报错提示金额不足，请充值,结束购买。
3. 金额够 创建产品（数据库信息），失败提示错误并结束购买，成功 产品accountUuid为null，调动支付接口。
4. 支付失败，过期时间立即设为当前。
5. 支付成功，accountUuid设置用户accountUuid，并下发命令。
6. 下发成功正常开通业务，下发失败由运维人员确定是开通，还是暂无法开通，并退款。
