消息：APICreateInterfaceMsg

权限：category = TunnelConstant.ACTION_CATEGORY, names = {"create"}

参数（默认必填）：
    客户：NASS不传，BOSS必传
    名称
    描述：非必填

    连接点：1.若选云
            从CloudVO中列出所有云类型,页面显示name,选择一个传参uuid(通用查询：APIQueryCloudMsg)
            再选择，cloudType为选择的云类型uuid,state为Enabled,(若NASS status为Open) 的连接点（通用查询：APIQueryEndpointMsg）
           2.若选IDC
            通过Province选择节点（查询：APIListProvinceNodeMsg | APIQueryNodeMsg）
            再选择该节点下，endpointType为ACCESSIN，state为Enabled,(若NASS status为Open) 的连接点（通用查询：APIQueryEndpointMsg）

    类型：即端口规格，列出所选连接点获取可用端口规格，显示name,传参uuid（查询：APIGetInterfaceTypeMsg）
    付费类型
    付费时长


请求消息体示例：
{
    "com.syscxp.header.tunnel.tunnel.APICreateInterfaceMsg":{
        "name":"陈升接入日本",
        "description":"测试",
        "endpointUuid":"38c68ae9705c45cda325cd07d7806b70",
        "portOfferingUuid":"SFP_1G",
        "duration":2,
        "productChargeModel":"BY_MONTH",
        "session":{
            "uuid":"b64a71a1b1a144a2899d67cffccf448d"
        }
    }
}

后台验证（validate）：
    1.判断同一个用户的接口名称是否已经存在
    2.判断账户金额是否充足

业务流程：
    1.根据连接点和接口类型策略分配端口
        分配限制：若是共享口：一个用户在同一个连接点下只能购买一个共享口
                 若是独享口：分配的端口没有被使用
    2.保存数据：
        accountUuid = null
        state = Unpaid
        expireDate = dbf.getCurrentSqlTime()
    3.调用支付：
        支付成功：
            记录生效订单
            accountUuid = ownerAccountUuid
            state = Paid
            expireDate = dbf.getCurrentSqlTime()+时长
        支付失败：
            不变
